# GitHub User Finder

Пасечник Игорь Павлович

## Описание

Приложение «GitHub User Finder» — это графический инструмент для поиска пользователей GitHub с возможностью сохранения избранных профилей.

**Основные функции:**
* поиск пользователей GitHub по имени;
* отображение результатов в табличном виде (логин, имя, компания, местоположение);
* добавление пользователей в избранное;
* сохранение избранного в JSON‑файл;
* просмотр списка избранных пользователей.

## Технические детали

**Технологии:**
* Python 3.x;
* tkinter — графический интерфейс;
* requests — работа с GitHub API;
* JSON — хранение избранных пользователей.
