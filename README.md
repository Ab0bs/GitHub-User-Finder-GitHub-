# GitHub User Finder

Пасечник Игорь Павлович

## Описание

Приложение для поиска пользователей GitHub с возможностью добавления избранных пользователей в список и их сохранения в JSON‑файл.

## Использование API GitHub

Приложение использует GitHub API для получения информации о пользователях. Запросы отправляются по адресу:
https://api.github.com/user/{username}


Где `{username}` — имя пользователя GitHub.

## Как использовать

1. Запустите приложение: `python main.py`.
2. Введите имя пользователя GitHub в поле ввода.
3. Нажмите кнопку «Найти» для отображения информации о пользователе.
4. Нажмите «Добавить в избранное», чтобы сохранить пользователя в списке избранных.
5. Избранные пользователи сохраняются в файле `favorites.json`.

## Примеры использования

**Тест 1: Поиск существующего пользователя**
- Введите: `octocat`
- Нажмите «Найти»
- Ожидаемый результат: отображение информации о пользователе

**Тест 2: Поиск несуществующего пользователя**
- Введите: `nonexistentuser12345`
- Нажмите «Найти»
- Ожидаемый результат: сообщение об ошибке «Пользователь не найден!»

**Тест 3: Добавление в избранное**
- Найдите существующего пользователя
- Нажмите «Добавить в избранное»
- Ожидаемый результат: пользователь появляется в списке избранных и сохраняется в `favorites.json`

## Зависимости

- Python 3.x
- `requests` (для работы с API)
- `tkinter` (встроенная библиотека Python для GUI)
